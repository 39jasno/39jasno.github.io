About this website project

This project involves transforming the current magnet checking process from a manual, Excel-based system into a streamlined web application with a centralized database. This new system enhances data accuracy, accessibility, and efficiency by allowing users to input, manage, and review magnet model specifications and inspection records through a user-friendly interface. By centralizing the data, the application eliminates issues related to version control, data loss, and inconsistent formatting, while also providing secure access, improved traceability, and better oversight for administrators. This upgrade modernizes their workflow and lays the foundation for more scalable and reliable quality control operations.

Synthesis of the Practicum Engagement

This project transformed a manual, Excel-based magnet inspection workflow into a centralized, web-based system, improving data accuracy, usability, and efficiency. Developed using Visual Studio Code and deployed on a local XAMPP server, the system addressed issues such as version control, inconsistent formatting, and accidental data loss.

Designed for touchscreen tablets, the interface is simple, responsive, and tailored for industrial environments. Features include real-time alerts, visual charts, and searchable history logs, enhancing both usability and decision-making.

The system supports role-based access, separating operators from administrators to maintain data security and streamline workflows. Key functions include data entry, model management, and inspection review.

Development challenges like data validation and UI complexity were addressed through testing and optimization. The final product is reliable, scalable, and ready for future enhancements like remote access and cloud deployment, marking a major step forward in production quality and digital transformation.
